# IDE Configuration

This document provides consolidated IDE configuration guidelines for various project types.

## File encoding and formatting

* COM-002 Line endings: Use LF line endings.

* COM-006 Trailing whitespace: Trim trailing whitespace on save.

## Indentation and spacing

* COM-003 Indentation: Use spaces only; do not use tabs or smart tabs. (Note: Specific indent size and tab width are defined per language.)
* COM-008 Continuation indent: Use continuation indent for wrapped lines. (Note: Specific continuation indent size is defined per language.)
* COM-009 Wrapping behavior: Do not wrap automatically while typing; prefer stable, explicit formatting decisions.
* COM-010 Visual guides: No visual guides configured; do not rely on guide columns.
